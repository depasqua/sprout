# SMS Integration

**As a** system administrator  
**I want to** send SMS reminders to volunteers  
**So that** I can reach them through multiple channels

**Acceptance Criteria:**
- System integrates with SMS API (MailChimp/SendGrid)
- SMS can be sent manually or automatically at configurable intervals
- SMS content can be customized
- SMS history tracked in volunteer profile
- System tracks SMS delivery and responses
