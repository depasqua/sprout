# Reporting & Data Export

**As a** system administrator  
**I want to** generate reports and export data  
**So that** I can analyze trends and use data in other systems

**Acceptance Criteria:**
- System can generate PDF reports comparing years
- Reports include general analytics (sign-ups, attendance, applications, etc.)
- Reports can be printed or exported
- Data can be exported electronically in common formats (CSV, Excel, JSON)
- Export includes all relevant volunteer data
- Export can be filtered by date range, status, etc.
